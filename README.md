# Projet-Olympique

Le dossier "projet_environnement_de_test" contient le projet sous un format adapter à un environnement de test avec une application telle que Laragon ou Wamp. 
Le dossier "projet_site_déployé" contient le projet sous un format adapter au déploiement de l'application sur une plateforme en ligne telle que Siteground.
